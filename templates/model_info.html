1. Dataset Source

The model is trained using the Brain Tumor MRI Dataset from Kaggle , which includes MRI scans labeled into four categories: glioma, meningioma, pituitary tumor, and no tumor.

2. Dataset Distribution

The dataset offers a balanced representation across categories, helping prevent model bias and ensuring reliable classification performance.

3. Model Architecture & Performance

The CNN architecture consists of multiple convolutional and pooling layers followed by fully connected layers. The model achieves over 97% accuracy on the validation set, with high precision and recall across all classes.

4. Confusion Matrix

The matrix shows that the model is highly effective in differentiating between tumor types with minimal misclassifications.
